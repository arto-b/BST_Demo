\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_queue}{Queue} class. 

This header declares the \doxylink{class_queue}{Queue} class, which is used as an explicit queue to support non-\/recursive operations within a binary search tree. Specifically, it is utilized by the level\+Order() traversal method.

Its purpose is to assist in tree traversal by visiting the nodes level-\/by-\/level, starting from the root.

Implementation details are defined in \doxylink{_queue_8cpp}{Queue.\+cpp}.
