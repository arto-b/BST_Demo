\doxysubsection{Detailed Description}
Internal node type used by the \doxylink{class_stack}{Stack} class. 

\doxylink{struct_stack_node}{Stack\+Node} stores a pointer to a tree \doxylink{class_node}{Node} and a link to the next stack node. This struct is used only to implement the stack\textquotesingle{}s internal linked list and does not assume ownership of the Node\texorpdfstring{$\ast$}{*} it stores.
