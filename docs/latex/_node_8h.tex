Declaration of the \doxylink{class_node}{Node} class. 

This header declares the \doxylink{class_node}{Node} class used to represent individual nodes within a binary search tree (\doxylink{class_b_s_t}{BST}). The \doxylink{class_node}{Node} class encapsulates a single integer data value along with pointers to its left and right child nodes.
