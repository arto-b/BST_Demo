\doxysubsection{Detailed Description}
Iterative binary search tree storing integer values. 

The \doxylink{class_b_s_t}{BST} class is an iterative, pointer-\/based binary search tree implementation using raw pointers and manual dynamic memory management. The \doxylink{class_b_s_t}{BST} owns all allocated nodes and is responsible for their cleanup through its destructor.

The \doxylink{class_b_s_t}{BST} class manages dynamically allocated \doxylink{class_node}{Node} objects, forming a binary search tree structure. All operations, including insert, search, delete, and traversal, are implemented iteratively to avoid recursion.

The tree maintains standard \doxylink{class_b_s_t}{BST} ordering properties\+:
\begin{DoxyItemize}
\item Left subtree contains values less than the node.
\item Right subtree contains values greater than the node.
\item Duplicate values are ignored.
\end{DoxyItemize}

Memory Management\+: The \doxylink{class_b_s_t}{BST} owns all its nodes. The destructor invokes a private destroy\+Tree() helper method, ensuring that all nodes are properly freed and preventing memory leaks.
