\doxysubsection{Detailed Description}
Represents a node within a binary search tree. 

The \doxylink{class_node}{Node} class represents the fundamental unit of an iterative, pointer-\/based binary search tree. Each node maintains self-\/referential raw pointers to its left and right child nodes and stores a single integer data value.

Encapsulation is enforced through data hiding. All data members are declared as private and may only be accessed or modified through public accessor methods.
