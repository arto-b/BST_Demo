\doxysubsection{Detailed Description}
Internal node type used by the \doxylink{class_queue}{Queue} class. 

\doxylink{struct_queue_node}{Queue\+Node} stores a pointer to a tree \doxylink{class_node}{Node} and a link to the next queue node. This struct is used only to implement the queue\textquotesingle{}s internal linked list and does not assume ownership of the Node\texorpdfstring{$\ast$}{*} it stores.
